Nicklas Larsson has produced new binaries for GRASS 8.5.0 (preview) releases. This an update to the newest development version of GRASS 8.5.

For any of you who would like to try to compile these on your own, Nicklas has now included build instructions in the GRASS source code so that anyone can pretty easily compile the latest version. See the updated download page for instructions on how to do this. 

Enjoy!
