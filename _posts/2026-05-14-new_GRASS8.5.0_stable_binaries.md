Nicklas Larsson has produced new binaries for GRASS 8.5.0 stable release.

Although they compile without issue, we have had reports of the Intel binary produced here may have trouble running on Intel Macs. If you have an old Intel Mac and run into problems, you might be able to compile it yourself have have it run for you.

For any of you who would like to try to compile 8.5.0 on your own, Nicklas has now included build instructions in the GRASS source code so that anyone can pretty easily compile the latest version. See the updated download page for instructions on how to do this.

Enjoy!
